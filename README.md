# Git-Installer(未完成-擱置中)
 Git的 安裝/設定/下載/推送 腳本  

# Use
在下載後你會看到兩個資料夾  
開啟set資料夾 編輯`userdata.json`  
之後開啟main.py

# json 格式
```json
{
    "name": "your name",
    "email": "your email",
    "project": "your project",
    "__example__": "https://github.com/XiaXia009/Git-Installer.git"
}
```
